# Enrollment-Predictor
##Requirements

•	Python 3.13 installed locally (optional, for local testing outside Docker)

•	Visual Studio Code (VS Code) installed

o	Recommended extensions: Python, Docker

•	Install Flask 
```pip install flask```

•	Docker Desktop installed and running

o	Download: https://www.docker.com/products/docker-desktop

o	Ensure Docker Engine is running (green light in Docker Desktop)

## Run Environment

• Build containers:
```docker-compose build```

• Start containers:
```docker-compose up```

• View logs:
```docker-compose logs -f```

• Stop containers:
```docker-compose down```
```docker-compose down -v```
